# Outputs Directory

This directory contains all generated outputs from the ML pipeline.

## Structure

- `models/` - Trained model files (.pkl, .pt, .h5)
- `logs/` - Training logs and metrics
- `predictions/` - Model predictions on test data
- `visualizations/` - Plots and figures

## Model Files

Models are saved with timestamps and version numbers for tracking.

## Logs

Training logs include:
- Loss curves
- Metric histories
- Hyperparameters
- System information
