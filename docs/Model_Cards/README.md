# Model Cards

This directory contains model cards for all trained models.

## Available Models

### Classic Models
- Random Forest
- XGBoost

### Neural Networks
- Multi-Layer Perceptron (MLP)
- LSTM
- Transformer
- Graph Neural Network (GNN)
- Variational Autoencoder (VAE)

### Reinforcement Learning
- Q-Learning

## Model Card Template

Each model card follows a standard format:
1. Model Overview
2. Intended Use
3. Training Data
4. Architecture Details
5. Performance Metrics
6. Limitations
7. Ethical Considerations
